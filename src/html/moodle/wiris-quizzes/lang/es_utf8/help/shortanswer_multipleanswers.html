Esta es una opción de WIRIS Quizzes para preguntas mejoradas de matemáticas y ciencia.

Para activar esta opción, Editor de fórmulas tiene que estar también activada. Si se utiliza, la solución deja de ser un sólo valor o fórmula. En su lugar, es un conjunto de parejas variable=valor. Estas variables deben estar entre las definidas en el algoritmo. El campo Respuesta pasa a ser una lista de variables separadas por espacio.

Visite manual de WIRIS Quizzes para más información.
