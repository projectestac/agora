Элементы задания

Для упрощения оценивания у задания Упражнение должно быть разумное число "оцениваемых элементов". Каждый элемент должен покрыть специфический аспект задания. Обычно у задания будет от 5 до 15 элементов для комментариев и оценивания, реальное число зависит от размера и сложности задания. Задания только с одним элементом тоже дозволены и имеют стратегию оценки, подобную стандартному заданию Moodle.

Стратегия оценки задания зависит от выбора способа оценки элементов:

Не оценивать: Элементы являются описаниями аспектов задания. Эксперта просят прокомментировать каждый из этих аспектов. Как и во всех стратегиях оценивания, есть область для общих комментариев.

Суммарно: У оцениваемых элементов есть следующие три признака:

  1. ОПИСАНИЕ элемента. Оно должно ясно устанавливать, какой аспект задания оценивается. Если оценка является качественной, полезно сообщить подробности того, что считается отличным, средним и плохим.
  2. ШКАЛА элемента. Есть некоторые предопределенные значения. Их диапазон: от простой шкалы Да/нет через шкалы с заданными значениями до полной 100% шкалы. У каждого элемента может быть своя собственная шкала, которая соответствует числу возможных изменений для этого элемента. Отметим, что шкала НЕ ОПРЕДЕЛЯЕТ важность элемента в вычислении итоговой оценки; у шкалы в 2 балла то же самое "влияние" как и у шкалы в 100 баллов, если у соответствующих элементов одинаковый вес.
  3. ВЕС элемента. По умолчанию элементам дают одинаковую значимость в вычислении итоговой оценки задания. Это можно изменить, назначив больше важным элементам вес больше 1, а менее важным - вес ниже 1. Изменение весов НЕ ВЛИЯЕТ на максимальную оценку, значение которой устанавливается параметром "Максимальная оценка". Весам можно назначить отрицательные значения, это - экспериментальный параметр.

Погрешность оценки: Элементы обычно описывают определенные пункты или аспекты задания. Оценивается присутствие или отсутствие этих элементов или аспектов. Учитель должен опубликовать заполненную таблицу, в которой предлагаются оценки для случаев, когда все элементы присутствуют, когда один отсутствует, когда два отсутствуют и т.д. Если определенные элементы более важны чем другие, тогда этим элементам можно дать вес больше 1. Незначительным элементам можно дать вес меньше 1. Полная "погрешность расчета" - это взвешенная сумма недостающих элементов. Эксперт всегда может сделать незначительную корректировку этих предложенных оценок.

Критерий: Элементы дают ряд "уровней" утверждений, которые используются в оценке задания. Утверждения могут быть совокупными или каждое может быть самостоятельным. Эксперт должен решить, какое утверждение лучше всего соответствует каждой работе. Учитель должен также связать каждый критерий утверждения с предложенной оценкой. Они должны быть в нормальном порядке. Эксперт может сделать незначительную корректировку этих предложенных оценок.

Рубрика: Это подобно оцениванию по Критерию в случае, когда имеется более одного критерия. Число критериев задано в параметрах задания. В пределах каждого критерия может быть до пяти "уровней" утверждений. В задании число уровней для каждого критерия может быть разным. В критерии пустое утверждение уровня сигнализирует о конце уровня. Таким образом, у некоторого критерия может быть 2 уровня, у другого - 3; (до 5 уровней). Критерии могут иметь вес. Уровни отмечаются 0, 1, 2, 3, 4. Значение оценки задания - взвешенная сумма этого множества.
