Imposer le téléchargement

Ce réglage permet de vous assurer que la ressource n'est pas incorporée dans une fenêtre du navigateur, mais téléchargée sur l'ordinateur du participant. Il est particulièrement utile si l'on veut éviter des embûches dues aux incompatibilités entre différents navigateurs et les mauvaises versions de plugins multimédias. C'est la solution la plus accessible pour fournir des ressources constituées de fichiers.
