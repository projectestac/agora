Rapport de participation

Module d'activité - Choisissez le type d'activité sur lequel vous voulez avoir un rapport (par exemple Chat, Forum, Test, etc.). Seuls les modules utilisés dans le cours seront affichés dans le menu déroulant.

Remonter - Choisissez la durée sur laquelle vous voulez le rapport.

Afficher uniquement - Choisissez le rôle dont vous voulez un rapport des activités (par exemple étudiant, enseignant, etc.).

Afficher les actions - Choisissez si vous voulez un rapport sur les affichages, les messages postés ou les deux (Toutes les actions).

Une fois les paramètres ci-dessus sélectionnés, cliquez sur le bouton Valider.

Une liste de toutes les activités du type sélectionné dans ce cours sera générée. Choisissez l'élément sur lequel vous voulez un rapport et cliquez sur le bouton Valider.

La définition des affichages et des messages pour le type d'activité sélectionné sera affichée.
par exemple :
Affichage : View Discussion, Search, Forum, Forums, Subscribers
Message : Add Discussion, Add Post, Delete Discussion, Delete Post, Move Discussion, Prune Post, Update Post

Les utilisateurs correspondant au critère Afficher uniquement seront affichés dans un tableau, avec leur nom, prénom et s'ils ont effectué l'action désignée (et combien de fois, le cas échéant). Vous pourrez alors utiliser cette liste pour envoyer un message à plusieurs participants en les sélectionnant et choisissant Ajouter/envoyer un message.
