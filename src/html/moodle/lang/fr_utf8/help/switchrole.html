Prendre le rôle

Ce menu vous permet de prendre temporairement un autre rôle afin de voir comment le cours se présente pour quelqu'un ayant ce rôle.

Les rôles disponibles sont les mêmes que ceux que vous pouvez attribuer à d'autres participants.

Lorsque vous avez terminé votre visite dans un autre rôle, vous pouvez revenir à votre rôle normal en cliquant sur le bouton dans la page de cours ou sur le lien en haut de la page.
