Aide sur l'évaluation : version étudiant

Sur cet écran, vous avez une vue d'ensemble des notes que vous avez obtenues pour ce cours.

  • Au milieu, vous verrez toutes les notes partielles obtenues pour chaque activité réalisée dans ce cours. Cliquez sur les noms et les notes partielles de chaque activité pour obtenir leur détail.
  • La colonne des totaux se trouve à droite de toutes les activités.
  • Dans l'entête se trouve un lien nommé « stats » qui renvoie, dans une nouvelle fenêtre, des statistiques basées sur tous les devoirs et/ou tests des participants de ce cours. Vous aurez des informations sur :
    • Note globale la plus haute : la somme la plus élevée de tous les points obtenus par un participant de ce cours.
    • Note globale la plus basse : la somme la plus basse de tous les points obtenus par un participant de ce cours.
    • Moyenne des notes globales : la somme de toutes les notes globales pour ce cours, divisée par le nombre de participants.
    • Médiane : la valeur centrale qui partage l'échantillon en 2 groupes de même effectif, 50% au-dessus et 50% en dessous.
    • Mode : la note atteinte par le plus grand nombre de participants.
    • Écart type : une mesure de la dispersion des résultats.
