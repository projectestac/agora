Cacher le lien de prévisualisation

Si cette option est active, le bouton « Prévisualiser » de la page d'activité d'un paquetage SCORM/AICC est caché.

Dans le cas contraire, l'étudiant peut choisir d'exécuter l'activité en mode « Prévisualisation » ou essayer de la réaliser normalement.
