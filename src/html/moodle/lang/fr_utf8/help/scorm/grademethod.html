Méthode d'évaluation des paquetages SCORM

Le résultat de l'activité sur un paquetage SCORM/AICC peut produire des notes dans le tableau d'évaluation global de Moodle selon plusieurs méthodes :

  • Objets complétés
    La note est basée sur le nombre d'objets d'apprentissage complétés/réussis de l'activité. La note maximale est le nombre d'objets d'apprentissage contenus dans le paquetage, quel que soit le nombre de tentatives.
  • Note la plus haute
    La note sera le plus haut score AICC obtenu par les élèves dans les objets d'apprentissage achevés.
  • Note moyenne
    La note sera dans ce cas la moyenne des scores AICC obtenus par les élèves dans les objets d'apprentissage achevés pour chaque tentative.
  • Note sommative
    Dans ce mode les scores AICC obtenus à chaque tentative sont ajoutés.
