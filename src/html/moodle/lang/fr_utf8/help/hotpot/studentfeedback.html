Feedback de l'étudiant

Si cette option est activée, un lien vers une fenêtre surgissante (popup) s'affichera lorsque l'étudiant cliquera sur le bouton « Vérifier ». Le feedback permet à l'étudiant d'adresser ses appréciations à l'enseignant concernant l'activité.

Cette option accepte les propositions ci-dessous.

Aucun
Aucun feedback ne s'affichera.

Page web
La fenêtre de feedback affichera une page web. Pour ce faire, il vous faut désigner l'adresse de la page à afficher, telle que par exemple : http://myserver.com/feedbackform.html.

Formulaire de feedback
la fenêtre de feedback affichera un formulaire permettant l'envoi automatisé d'un courriel (grâce à un script formmail.pl par exemple). Vous devrez entrer l'adresse du script à utiliser, par exemple : http://myserver.com/cgi-bin/formmail.pl.

Forum Moodle
Affiche les sujets du forum de ce cours.

Message Moodle
Affiche la fenêtre de messagerie instantanée de Moodle. Si le cours comporte plusieurs enseignants, l'étudiant devra choisir l'un d'eux avant l'affichage de la messagerie.
