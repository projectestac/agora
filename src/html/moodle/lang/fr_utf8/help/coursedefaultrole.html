Rôle par défaut

Ce rôle est celui qui est attribué aux utilisateurs lorsqu'aucun autre rôle ne leur est explicitement attribué, par exemple lorsqu'ils s'inscrivent individuellement à votre cours avec une clef d'inscription.

Ce rôle est aussi utilisé comme défaut par les méthodes d'inscriptions se basant sur les données de systèmes externes pour l'inscription des participants à un cours.

La plupart du temps, ce réglage devrait rester celui du rôle par défaut pour tout le Moodle., afin que l'administrateur puisse décider du rôle par défaut de tous les cours.
