Visibilité des champs du profil

Chaque champ personnalisé du profil peut avoir l'un des trois réglages de visibilité suivant : caché, visible pour tous et visible pour l'utilisateur.

Le réglage « caché » est choisi par l'administrateur désirant conserver des données privées des utilisateurs. Le réglage « visible pour l'utilisateur » sera utilisé pour un champ contenant des données personnelles, tandis que « visible pour tous » sera utilisé pour tout autre type d'information.
