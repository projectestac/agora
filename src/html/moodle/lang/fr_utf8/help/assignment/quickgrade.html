Évaluation rapide

Lorsque l'évaluation rapide est activée, il est possible d'évaluer rapidement plusieurs devoirs affichés sur la même page.

Modifiez simplement les évaluations et les commentaires, puis utilisez le bouton Enregistrer au bas de la page pour enregistrer d'un coup toutes les modifications effectuées sur cette page.

Les boutons d'évaluation normale continuent bien sûr à fonctionner de la même manière, pour le cas où vous avez besoin de plus de place. Votre préférence d'évaluation rapide sera enregistrée et s'appliquera à tous les devoirs de tous les cours.
