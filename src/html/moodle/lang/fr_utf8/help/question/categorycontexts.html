Contextes des catégories de questions

Dans Moodle 1.9, les catégories de questions sont séparées en divers contextes. Chacun de ces contextes possède une hiérarchie séparée des catégories de questions. Les contextes des catégories de questions sont :

  • le contexte de l'activité : les questions ne sont disponibles que dans un module d'activité ;
  • le contexte du cours : les questions sont disponibles dans tous les modules d'activité d'un cours et dans le cours, à partir du lien « Questions » dans le bloc d'administration du cours ;
  • les contextes de catégories de cours : les questions sont disponibles dans tous les modules d'activité et tous les cours de la catégorie. Puisqu'une catégorie de cours peut contenir d'autres catégories, vous pouvez partager vos questions dans l'une ou toutes les catégories parentes ;
  • le contexte système : les questions sont disponibles dans tous les cours et toutes les activités de votre installation de Moodle.

Vous ne pourrez voir et utiliser les contextes au-dessus du niveau du cours que si l'administrateur de votre Moodle vous a donné l'autorisation de le faire.
