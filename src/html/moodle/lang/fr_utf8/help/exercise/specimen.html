Formulaire d'évaluation type

Cette page montre le détail du formulaire qui sera utilisé pour les évaluations. Vous et votre enseignant l'utiliserez pour juger votre travail.

Les notes que vous modifierez sur ce formulaire ne seront pas enregistrées. Il s'agit ici d'un spécimen, mais le formulaire effectif d'évaluation que vous aurez à remplir lors de l'exercice y ressemblera très fortement.
