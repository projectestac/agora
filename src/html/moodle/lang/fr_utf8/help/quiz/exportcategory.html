Exporter la catégorie

Le menu déroulant Catégorie est utilisé pour sélectionner la catégorie dont il faut tirer les questions à exporter.

Certains formats d'importation (formats GIFT et XML) permettent l'inclusion de la catégorie dans le fichier d'exportation, donnant ainsi la possibilité de la recréer au cours de l'importation. Pour que ces données soient incluses dans le fichier, veuillez cocher la case Vers le fichier.
