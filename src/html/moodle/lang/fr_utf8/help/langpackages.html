Paquetages de langue

Paquetage de langue principal anglais
Ce paquetage est la source primaire pour toutes les chaînes de caractères faisant partie intégrante de Moodle. Ce paquetage est distribué avec Moodle et ses extensions. Il est constitué de fichiers PHP comprenant des chaînes de caractères et de fichiers d'aide en format HTML, le tout dans le dossier « lang/en_utf8 ». Toutes les extensions additionnelles de Moodle (modules, blocs, etc.) doivent fournir leurs propres fichiers principaux en anglais. Le paquetage principal anglais ne peut pas être modifié à partir de l'interface de Moodle.
Paquetage de langue principal
Ce paquetage est la traduction officielle du paquetage principal anglais. En principe, il est entretenu et développé par un membre volontaire de la communauté Moodle. Les paquetages de langue principaux de Moodle peuvent être mis à jour automatiquement à partir de l'administration du Moodle, dans le menu « Langue » (vous y trouverez un utilitaire vous permettant de choisir, télécharger et installer ces paquetages de langue directement depuis Moodle). Ils peuvent être également téléchargés depuis http://download.moodle.org.
Modifications locales
Les modifications locales des paquetages de langue principaux sont enregistrées dans un dossier séparé de votre serveur. De cette manière, vous pouvez effectuer vos propres modifications aux langues de Moodle sans effacer les paquetages officiels.
