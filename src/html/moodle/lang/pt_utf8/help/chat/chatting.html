Utilização do Chat

O módulo de chat inclui algumas características para tornar o chat mais agradável.

Carinhas
Qualquer carinha (emoticon) que possa usar em outras partes do Moodle poderá ser usada também aqui e será substituída pelo gráfico correspondente. Por exemplo, :-) =
Vínculos
Os endereços da Web serão transformados automáticamente em vínculos.
Personalização
Pode começar uma linha com "/me" ou ":" para personalizar a mensagem. Por exemplo, se o seu nome form Joaquim, e escrever ":sorri!" ou "/me sorri!", todos os outros participantes verão "Joaquim sorri!"
Alarmes
Pode enviar um som para os outros participantes, clicando no vínculo "beep" junto do seus nomes. Um atalho útil para enviar um som para todos os participantes consiste em escrever "beep all".
HTML
Se souber algum código HTML, poderá usá-lo no seu texto para fazer coisas como, por exemplo, inserir imagens, tocar sons, ou criar texto com cores diferentes.
