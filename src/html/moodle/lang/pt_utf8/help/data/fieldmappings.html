Corresponências entre campos

Este menu permite-lhe manter os dados da base de dados existente. Para preservar os dados num campo, deverá faze-lo corresponder a um campo novo, onde aparecerão os dados. Qualquer campo pode também ser deixado em branco, sem que se copie nenhuma informação para ele. Qualquer campo antigo que não seja feito corresponder a um campo novo, será perdido e os seus dados retirados.

Pode criar unicamente correspondencias entre campos do mesmo tipo, de maneira que cada menu desplegável terá diferentes campos. Também deverá ter cuidado para não associar um campo antigo para mais do que um campo novo.
