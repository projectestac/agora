Tipos de Trabalho

Há vários tipos de trabalho disponíveis e continuam a adicionarem-se mais

Actividade fora de linha

Este tipo de trabalho é útil quando o trabalho for realizado fora do Moodle. Poderá ser algo algures na Web, ou um trabalho cara a cara.

Os alunos podem ver a descrição do trabalho, mas não poderão enviar ficheiros nem nenhuma outra coisa. A avaliação funciona normalmente, e os alunos receberão notificações das suas notas.

Envio de um único ficheiro - este tipo de trabalho permite que cada aluno envie um ficheiro de qualquer tipo. Pode ser um documento de um processador de texto, uma imagem, um sítio Web comprimido ou algo que lhes for pedido aos alunos para
