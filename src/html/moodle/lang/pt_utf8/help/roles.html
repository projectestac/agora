Cargos

Um cargo é uma colecção de permissões definidas para o todo o sítio, que pode ser atribuído a utilizadores específicos em contextos específicos.

Por exemplo, poderá ter um cargo designado de "Professor", com definições que permitem que os professores façam determinadas coisas e não outras (se estiver a usar um pacote linguístico, em algumas versões do Moodle esse cargo poderá aparecer como "Teacher", pois o nome e descrição original dos cargos está na base de dados e não no pacote linguístico. O administrador do sítio deverá fazer a tradução). Uma vez esse cargo exista, poderá ser atribuído a alguém numa disciplina, tornando-o "Professor" para essa disciplina. Poderia igualmente atribuir esse cargo a um utilizador numa categoria de disciplinas, para que seja "Professor" em todas as disciplinas nessa categoria, ou atribuir esse cargo a um utilizador apenas num único fórum, dando a esse utilizador as competências do professor, apenas nesse fórum.

Um cargo deve ter um nome. Se precisar nomear o cargo para múltiplas línguas poderá optar por usar a sintaxe multi-línguas, tal como 

<span lang="en">Teacher</span>
<span lang="pt">Professor</span>
Se usar essa definição deverá verificar que o filtro de "Conteúdo Multilinguístico" está activo no seu sítio.

O Nome curto é necessário para outros plugins do Moodle que podem precisar de se referir aos cargos (por exemplo ao carregar utilizadores de um ficheiro ou fazer inscrições com o plugin de inscrição).

A Descrição é simplesmente para descrever o cargo brevemente, de modo a que todos compreendam a natureza do cargo.
