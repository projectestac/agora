Version 2.0 
===============================================================================
IMPORTANT NOTICE:
===============================================================================
The permission component changed in v2.0 from pn_bbode to bbcode, please update
your permission rules!!!
===============================================================================
Upgrade from pn_bbcode:
* remove old pn_bbcode module in the filesystem, do not remove the module in
  Zikula!
* copy new bbcode files to your modules folder
* regenerate modules list and continue upgrading as usual
===============================================================================
  - This version needs min. Zikula 1.0 or later to run, do not even try it 
    with PostNuke .764 or older - it won't work
  - renamed to bbcode to get rid of the pn_ prefix.
  - added some config options
  - added [#4596] Lightbox, credits for the code go to soultrain, see
    http://support.pn-cms.de/module-CMS_Support_Forum-viewtopic-topic-26109.htm
  - added [#2205] spoiler tag! Uses the same color for text and background, to be defined
    in the bbspoiler css class. To read the text, highlight it using the mouse.
  - added [noparse][/noparse] to avoid parsing bbcode
  - updated GeSHi to 1.0.7.20
  - added feature: mimetex-support.

Version 1.17: May 20, 2005
    - fixed problem with empty code blocks
    - special support for mail2forum feature of pnForum
    - admin can allow access to links (url and mail) with permissions:
      unreg. users (group) | bbcode:modname:Links | .* | ACCESS_NONE
      unreg. users (group) | bbcode:modname:Emails | .* | ACCESS_NONE
      e.g.
      unreg. users (group) | bbcode:pnForum:Links | .* | ACCESS_NONE
      unreg. users (group) | bbcode:pagesetter:Emails | .* | ACCESS_NONE
	  code tags will be added at the cursor position now, marked textportions will be empbraced
	  with tags
	  IMPORTANT: the textarea needs an id now which has to be passed to the bbcode module:
	  <!--[ pnmodfunc modname=bbcode type=user func=bbcodes textfieldid='mytextfieldid' ]-->
	  shows the bbcode selection
