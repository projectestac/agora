Release notes for GeoGebra Module for Moodle (http://moodle.org/plugins/pluginversions.php?plugin=mod_geogebra)

More information on each of the fixes can be found in the project 
development home at https://github.com/projectestac/moodle-mod_geogebra

Changes in progress (Moodle 2 version)
---------------------------------------------------------------------------------------
- At the moment, only works:
  * Create GGB activities 
  * View GGB activities with students mode recovering status
  * Save/Update attempts (from students view)
  * Backup/Restory activities (even from Moodle 1.9)
  * View current student attempts (pending adapt function to show all students when current user can grade)

- Pending:
  * Add new tab for "Preview" results (currently there are showed in "View" tab)
  * Show number of attempt (or, at least, a message when is the last one)
  * Review format of the attempt summary information (to add bold to the titles, for instance)

Changes in progress (Moodle 1.9 version)
---------------------------------------------------------------------------------------
- Fixed some Oracle bugs related with ` character
