This is a WIRIS quizzes feature for enhanced math and science questions.

This field is optional. If used, the field value must be the identifier of a function defined by the teacher in the algorithm. This function takes for argument the student's response and can return a boolean value (true implies correct and false incorrect) or a number between 0 and 1. This response will be the student's grade for this question.

Please go to WIRIS quizzes user manual for more information.
