Esta es una opción de WIRIS Quizzes para preguntas mejoradas de matemáticas y ciencia.

Para activar esta opción, Editor de fórmulas tiene que estar también activada. Si se activa, aparecerá en el formulario de respuesta un applet WIRIS CAS donde el alumno podrá efectuar cálculos auxiliares.

Visite manual de WIRIS Quizzes para más información.
