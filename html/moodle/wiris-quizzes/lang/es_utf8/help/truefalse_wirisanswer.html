Esta es una opción de WIRIS Quizzes para preguntas mejoradas de matemáticas y ciencia.

Este campo es opcional. Si se usa, su valor es el nombre de una variable. Esta variable tiene que ser de tipo booleano (cierto o falso). Habitualmente será el resultado de la evaluación de una condición. La respuesta correcta es entonces el valor de esta variable.

Visite manual de WIRIS Quizzes para más información.
