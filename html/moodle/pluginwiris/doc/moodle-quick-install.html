1        WIRIS Plugin for moodle quick installation

1.1      Uncompressing the package

Download WIRIS Plugin and unzip it on your moodle’s root directory.

1.2      Checking rights

Check that PHP user has read and write rights on pluginwiris directory, filter directory and lib/weblib.php file.

1.3      Installing

Open http://your_server/moodle_path/pluginwiris/install.php on your web browser and follow the steps.

1.4      Troubleshooting

If you have any problem using this quick installation, you can use manual WIRIS Plugin for moodle installation or send an e-mail to info@wiris.com.
