Télécharger l'analyse

Vous pouvez télécharger les données affichées à l'écran pour une analyse ultérieure. Trois types de fichiers sont possibles pour les donnée téléchargées.
Dans chaque cas, les données seront représentées dans un tableau avec les intitulés de colonne appropriés. Si le tableau est paginé, les données de toutes les pages seront téléchargées dans un seul fichier.

Format Excel

Vous obtiendrez un document .xls (Excel).

Format OpenOffice Writer

Les données seront présentées dans un tableau dans un fichier texte .sxw (OpenOffice).
Ce format n'est disponible que si la bibliothèque PHPWriter est installée dans le dossier /moodle/lib.

Format Texte

Dans ce cas, les données seront téléchargées dans un fichier texte pur. Chaque rangée du tableau sera sur une ligne différente, les données seront séparées par des tabulations.
