Type de combinaison des évaluations des forums

Le type de combinaison définit comment les évaluations données aux messages d'un forum sont combinées pour constituer la note finale (pour chaque message et pour toute l'activité forum).

Veuillez choisir entre les types de combinaison ci-dessous.

  • Moyenne des évaluations (défaut)

    La moyenne de toutes les évaluations données aux messages de ce forum. Ce type est particulièrement utile pour l'évaluation par les pairs lorsqu'un grand nombre d'évaluations sont effectuées.

  • Nombre d'évaluations

    Le nombre des messages évalués constitue l'évaluation finale. Utile lorsque le nombre de messages est important. Attention : ce nombre ne peut excéder la note maximale allouée pour le forum.

  • Évaluation maximale

    L'évaluation la plus élevée constitue l'évaluation finale. Ce type de combinaison permet de mettre en évidence la meilleure contribution des participants, leur permettant de poster un message de haute qualité, et des messages plus habituels comme réponses à d'autres participants.

  • Évaluation minimale

    L'évaluation la plus basse constitue l'évaluation finale. Ce type de combinaison promeut une culture de qualité pour chacun des messages du forum.

  • Somme des évaluations

    Toutes les évaluations d'un participant sont additionnées pour former l'évaluation finale. Attention : ce nombre ne peut excéder la note maximale allouée pour le forum.
