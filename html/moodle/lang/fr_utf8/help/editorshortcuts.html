Raccourcis clavier de l'éditeur WYSIWYG

Les fonctionnalités suivantes sont accessibles via les raccourcis clavier suivants :

Ctrl-O Modifier la police de caractère
Ctrl-P Modifier le corps des caractères
Ctrl-H Modifier le format de texte (titres, etc.)
Ctrl-1 to Ctrl-6 Modifier le niveau de titre
Ctrl-= Modifier la langue du texte pour les lecteurs d'écran ou spécifier une langue pour le filtre multilingue (ces langues apparaissent après l'option « Multi » dans le menu).
Ctrl-B Texte gras
Ctrl-I Texte italique
Ctrl-U Texte souligné
Ctrl-S Texte barré
Ctrl-, Texte en indice
Ctrl-. Texte en exposant
Ctrl-0 Nettoyer le HTML de Word
Ctrl-Z Annuler
Ctrl-Y Répéter
Ctrl-L Texte aligné à gauche
Ctrl-E Texte centré
Ctrl-J Texte justifié
Ctrl-/ Texte de gauche à droite
Ctrl-| Texte de droite à gauche
Ctrl-; Diminuer le retrait
Ctrl-' Augmenter le retrait
Ctrl-G Modifier la couleur du texte
Ctrl-K Modifier la couleur de fond
Ctrl-F Rechercher et remplacer
Ctrl-` Passer en mode code HTML
Ctrl-M Passer en mode plein écran
Ctrl-Alt-O Insérer une liste ordonnée
Ctrl-Alt-U Insérer une liste non ordonnée
Ctrl-Alt-R Insérer un filet horizontal
Ctrl-Alt-A Créer un ancrage
Ctrl-Alt-L Créer un lien
Ctrl-Alt-D Supprimer le lien
Ctrl-Alt-N Désactiver les liens automatiques
Ctrl-Alt-I Insérer une image
Ctrl-Alt-T Insérer un tableau
Ctrl-Alt-S Insérer une binette
Ctrl-Alt-C Insérer un caractère spécial

Limitations actuelles :

  • Le raccourci clavier Ctrl-` ne peut pas être utilisé pour repasser en mode WYSIWYG à partir du code HTML.
  • Il n'y a pas de raccourci pour les opérations dans les tableaux (disponibles seulement en mode plein écran). Elles ne sont de toute manière que peu accessibles sur le clavier, puisqu'elles dépendent de la sélection de cellules de tableaux.
