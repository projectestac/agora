Filtre texte

Ce filtre vous permet de sélectionner des informations sur la base d'un texte libre. Les options suivantes sont disponibles :

  • contient - cette option sélectionne uniquement les informations qui contiennent le texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • ne contient pas - cette option sélectionne uniquement les informations qui ne contiennent pas le texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • est égal à - cette option sélectionne uniquement les informations qui sont égales au texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • commence par - cette option sélectionne uniquement les informations qui commencent par le texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • se termine par - cette option sélectionne uniquement les informations qui se terminent par le texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • est vide - cette option sélectionne uniquement les informations égales à la chaîne de caractères vide (le texte saisi est ignoré)
