Création automatique de comptes utilisateurs

Les données d'inscription IMS Enterprise décrivent un ensemble d'utilisateurs. Si ce réglage est activé, les comptes utilisateurs peuvent être créés automatiquement pour tous les utilisateurs qui ne sont pas trouvés dans la base de données de Moodle.

On recherche dans la base de données les utilisateurs dans un premier temps d'après leur identifiant (« idnumber »), puis par leur nom d'utilisateur Moodle.

Mots de passe

Les mots de passe ne sont pas importés par la méthode d'inscription IMS Enterprise. Il est recommandé d'utiliser les méthodes d'authentification de Moodle pour authentifier les utilisateurs.
