Catégorisation automatique

Si l'élément <org><orgunit> est présent dans les données arrivant pour un cours, son contenu sera utilisé pour indiquer une catégorie, si le cours doit être créé à partir de rien.

La méthode d'inscription ne modifiera pas la catégorie des cours déjà existants.

Si aucune catégorie du nom désiré n'existe, une catégorie cachée portant ce nom sera créée.
