Télécharger les données en tableau

Il est possible de télécharger les données affichées dans le tableau à l'écran pour des analyses approfondies. Vous pouvez les télécharger dans les différents formats de fichier indiqués ci-dessous.

Dans chaque cas, les données seront présentées dans un tableau avec les titres des colonnes adéquats, comme à l'écran. Le tableau téléchargé :

  • n'est pas paginé. Toutes les données de toutes les pages sont téléchargées dans un seul fichier ;
  • n'est pas filtré suivant les initiales des noms des utilisateurs (certaines tables permettent de n'afficher qu'un sous-ensemble des utilisateurs en les sélectionnant d'après l'initiale des noms ou prénoms des utilisateurs) ;
  • n'est pas affecté par les tris que vous pouvez effectuer sur le tableau affiché à l'écran.

Feuille de calcul Microsoft Excel

Une feuille de calcul .xls sera téléchargée.

Feuille de calcul OpenDocument (ODS)

Une feuille de calcul .ods spreadsheet sera téléchargée. Ce format peut être ouvert à l'aide des logiciels OpenOffice.org, KOffice, Google Docs, NeoOffice, Zoho, IBM Lotus Symphony et Corel WordPerfect Office X4.

Fichier texte avec séparateur tabulation

Dans ce cas, un fichier texte pur sera téléchargé, comprenant une ligne pour chacune des rangées du tableau, avec les données de chaque colonne séparées par des tabulations. Un tel format peut être interprété par de nombreuses applications.

Fichier texte avec séparateur virgule (CSV)

Dans ce cas, un fichier texte pur sera téléchargé, comprenant une ligne pour chacune des rangées du tableau, avec les données de chaque colonne séparées par des virgules. Un tel format peut être interprété par de nombreuses applications.

Fichier XHTML

Dans ce cas, un fichier xhtml enregistrable et imprimable sera téléchargé, sans les entête et pied-de-page présents habituellement sur les pages de Moodle.
