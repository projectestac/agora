Définir les Préférences

Les paramètres suivants du carnet de notes sont disponibles à partir de la page « Définir les Préférences ».

  • Utiliser les fonctionnalités avancées : cela active/désactive les fonctions avancées du carnet de notes. Le mode normal affichera uniquement les points et les totaux sans catégories ni calculs spéciaux d'évaluation.
  • Afficher l'évaluation pondérée : définit si oui ou ou non les pourcentages pondérés seront affichés. Vous pouvez également définir si oui ou non les étudiants les verront.
  • Afficher les points : définit si oui ou non les points sont affichés. Les paramètres étudiant séparément de la vue instructeur.
  • Afficher les pourcentages : définit si oui ou non un pourcentage est affiché. Les paramètres étudiant séparément de la vue instructeur.
  • Afficher l'évaluation par lettre : définit si oui ou non une évaluation par lettre est affichée pour le total du cours.
  • Évaluation par lettre : définit comment l'évaluation par lettre est calculée soit en utilisant le pourcentage brut ou le pourcentage pondéré
  • Actualisation des entêtes : définit la fréquence avec laquelle les entêtes de colonne sont actualisés. Cela peut permettre de suivre une classe comportant de nombreux étudiants.
  • Afficher les éléments cachés : cela affichera ou masquera les éléments qui sont cachés. Cela ne s'applique qu'à la vue instructeur. Les étudiants ne verront pas les entrées du carnet de notes pour les éléments qui ne sont pas visibles pour eux. Les totaux des étudiants inclueront les éléments cachés si ce paramètre vaut « Oui ».
