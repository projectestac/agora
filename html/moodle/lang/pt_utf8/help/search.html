Como fazer uma pesquisa

A pesquisa de um texto aceita várias opções, apresentadas na lista abaixo. Pode combinar essas opções para tornar a sua pesquisa mais específica.

procurar estas palavras Para procurar uma ou mais palavras em qualquer parte do texto, simplesmente escreva as palavras, separadas por espaços. Todas as palavras com mais do que duas letras serão usadas.
+procurar +estas +palavras O exemplo da línea anterior também buscaria no texto palavras como "emprestasse", porque essa palavra contém o termo de pesquisa "estas". Para fazer com que sejam procuradas as palavras exactas, adicione um sinal mais no início.
+máquina -café Use o sinal menos para indicar palavras que não quer que sejam incluidas na pesquisa.
"máquina de café" Para procurar uma frase exacta, coloque-la dentro de aspas duplas.
user:Joaquim Para procurar texto criado por um utilizador em particular, use o prefixo "user:" seguido de algum dos nomes ou apelidos do utilizador./td>
userid:6 Se souber o número de identificação de um utilizador em particular, poderá procurá-lo nesta forma.
subject:relatório Para buscar uma palavra unicamente dentro do campo de assunto ou título do texto, escreva o prefixo "subject:" antes da palavra.

Para fazer uma pesquisa avançada, clique no botão para iniciar a pesquisa, sem escrever nada no campo de pesquisa. Aparecerá um formulário completo que permite fazer pesquisas mais complexas em forma mais simples.
