Relatório de Participação

Módulo de actividade - Seleccione o tipo de actividade em que deseja fazer o Relatório (exemplo: Chat, Fórum, Teste, etc.). Note que só os módulos usados na disciplina vão ser listados neste menu.

Olhar para trás - Seleccione o período que deseja incluir no relatório.

Mostrar só- Escolha se quer incluir no relatório a actividade dos alunos ou dos professores.

Acções a mostrar - Escolha se quiser o relatório das vezes que os particpantes viram as actividades, ou das vezes que enviaram alguma coisa, ou ambas acções (Todas as acções).

Após ter introduzido valores para os parâmetros acima, clique em Executar.

Será gerado um menu com todas as ocorrências nesta disciplina do módulo de actividade seleccionado nesta disciplina. Escolha uma actividade e clique em Executar.

A definição das acções incluídas em Vista e Envio, para o tipo de actividade seleccionada, serão dadas no ecrã.
Por exemplo:
Fórum Vistas: view discussion, search, forum, forums, subscribers.
Fórum Envios: add discussion, add post, delete discussion, delete post, move discussion, prune post, update post

Os utilizadores com o cargo seleccionado para Mostrar (aluno ou professor) serão listados no formato tabular com Nome, Número de ID e se (e quantas vezes) realizaram a acção procurada (Sim (núm) ou Não). Pode usar igualmente o relatório de participação para enviar uma mensagem de correio electrónico seleccionando os utilizadores destinatários e escolhendo Adicionar / enviar mensagem.
