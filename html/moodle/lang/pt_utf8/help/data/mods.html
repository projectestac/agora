Base de Dados

O módulo de base de dados permite ao professor, e/ou alunos, criar, mostrar ou pesquisar registos num banco de dados sobre um tema qualquer. O formato e estrutura desses registos poderá ser quase ilimitado, incluindo, entre outros, imagens, ficheiros, URL, números e texto. Poderá ter familiaridade com tecnologias semelhantes, se já tiver usado bases de dados Access da Microsoft ou Filemaker.
