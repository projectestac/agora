Importar a partir de ficheiro CSV

CSV quer dizer Valores Separados por Vírgulas e é um formato usado comumente para trocas de textos.

O formato esperado do ficheiro é um ficheiro de texto plano com uma lista de nomes de campos no primeiro registo. Os dados vão a seguir, com um registo por linha.

O separador dos campos é por omissão a vírgula, e por omissão os campos não serão colocados entre aspas ou outro caracter para delimitar campos.

Os registos deverão estar separados por caracteres de fim de linha (gerados usualmente quando se carrega na tecla de ENTER no seu editor de texto). Os caracteres de tabulação podem ser especificados com a sequência \t e uma nova linha com \n.

Exemplo de ficheiro CSV:

  nome,altura,peso
  Ana,180cm,80kg
  Alda,170cm,60kg
  Adriana,190cm,70kg

Advertência: é possível que todos os tipos de campos não sejam suportados.
