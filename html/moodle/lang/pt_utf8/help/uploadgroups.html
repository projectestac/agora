Enviar ficheiro de grupos

Esta facilidade permite enviar para o Moodle um ficheiro com a definição de grupos.

  • Cada linha no ficheiro deverá conter um registo
  • Cada registo consiste numa série de dados separados por vírgulas
  • O primeiro registo no ficheiro é especial, e contém uma lista de nomes dos campos, usada para definir o formato do resto do ficheiro.

    Campos necessários: os campos seguintes deverão ser incluidos no primeiro registo, e definidos para cada utilizador.

    groupname

    Campos por omissão: estes são opcionais. Se não forem incluidos, os seus valores serão obtidos a partir da língua em uso e da disciplina actual.

    idnumber, coursename, lang

    Campos opcionais: todos estes são completamente opcionais.

    description, enrolmentkey, picture, hidepicture

  • As vírgulas dentro dos dados deverão ser codificadas como &#44 . O script descodifica-las-á automaticamente.
  • Coursename será o nome curto da disciplina.
  • Nota: se um grupo já tiver sido registado na base de dados do Moodle para uma disciplina em particular, este script devolverá o nome desse grupo. Aos professores só é permitido enviar ficheiros de grupos nas disciplinas nas que tiverem autorização de editar.

A seguir vem um exemplo de um ficheiro válido:

groupname,idnumber,lang,description,picture
grupo1, Fís101, pt_utf8, este grupo precisa de atenção especial!, 0
grupo2, Mat243, , ,
